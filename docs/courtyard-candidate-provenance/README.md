# Courtyard — Candidate Provenance Records

One markdown provenance record per asset candidate, filled **before** the candidate is reviewed.

- **Template:** copy `../courtyard-asset-candidate-provenance-template.md`.
- **Naming:** `<character>-run<N>-candidate-<NN>.md` — e.g. `ari-run1-candidate-01.md`, `eli-run1-candidate-01.md`, `tara-run1-candidate-01.md`.
- **Rule:** if licence / rights / provenance are unclear, mark the candidate **blocked** — it is not a valid candidate.

> Recording provenance is **not** approval. Previewing a candidate is **not** approval.
> Only a separate governed, Tara-confirmed review can assign **approved asset** status.
