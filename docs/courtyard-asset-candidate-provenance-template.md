## Field notes (guidance)

- **Candidate ID** — stable unique id (e.g. `ari-r1-001`).
- **Character** — `Ari` / `Eli` / `Tara`.
- **Variant / round** — e.g. `draft`, `fixed`, `regen round 1`.
- **Source type** — e.g. AI-generated / marketplace / scan / sculpt / stock.
- **Licence / rights fields** — must be complete and compatible before review; unknown or incompatible rights = halt (Phase 1D §8).
- **Default pose** — should be an upright neutral stand; flag locomotion/action poses.
- **Scale / Grounded** — target ~1.75–1.9 m, feet at floor.
- **Blender clinic applied** — note structural fixes only (scale/ground/pose/anim); the clinic cannot fix identity or baked textures.
- **Preview result** — observations from `/courtyard/3d-preview` (incl. debug-grey vs textured, exposure used).
- **Review outcome** — one of `rejected` / `needs revision` / `visual candidate` / `draft asset`. **Never** `approved asset` here — that is a separate governed Tara-confirmed step.
