# Library / RAG Authority Contract

Reading is not remembering.

Retrieval is not Memory.

RAG context is not lived continuity.

Only `canonical_memory` derived from `archive_items.canonical_status = 'canonical'` may be spoken as lived continuity.

All other retrieved material requires attribution.

`canonical_memory` is a retrieval/speech classification label.

It is not a second Memory authority field.

It must always be derived from existing canonical Archive Memory.

Invalid `canonical_memory` labels must be downgraded before prompt injection.

The Library/RAG layer may classify retrieved context, but it must not canonise Memory.

Promotion into Memory must go through the existing Memory Review / archive governance path.

There is one crown: `canonical_status = canonical`.
